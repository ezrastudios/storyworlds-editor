# Story Worlds Editor

A visual world-building editor for designing consistent locations, environments and scenes for storytelling and AI illustration.

Story Worlds Editor is not a game. It is a continuity tool for creating reusable fictional places.

The goal is to help authors, artists and AI-assisted storytellers keep environments consistent across scenes, chapters and illustrations.

## Current direction

This project starts simple on purpose.

We will first build a small browser-based prototype using HTML, CSS, JavaScript and Three.js. As the project grows and the needs become clearer, we may later move to a more advanced stack.

## Core idea

A project contains:

- A world
- Places
- Scenes
- Cameras
- Notes
- Visual reference data

The map is the source of truth. Final illustrations can be generated later using the world as reference.

## Development philosophy

Each version should answer one design question.

Examples:

- How should terrain editing feel?
- How should a forest be painted?
- How should a place be named?
- How should a scene remember camera and lighting?

The prototype is allowed to be simple. Clarity matters more than visual perfection at this stage.
