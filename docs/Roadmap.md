# Roadmap

This roadmap focuses on learning and product clarity. Each version should validate one idea.

## Milestone 0 — Project Foundation

Goal: create the clean foundation for the project.

- Repository created
- Documentation added
- Static starter app added
- Project purpose defined

## Milestone 1 — Core Canvas

Goal: understand the 3D stage.

Features:

- Three.js scene
- Camera
- Renderer
- Basic terrain plane
- Orbit controls

Learning goal:

- What is a scene?
- What is a camera?
- What is a mesh?

## Milestone 2 — Terrain Editing

Goal: make the world editable.

Features:

- Raise terrain
- Lower terrain
- Smooth terrain
- Flatten terrain
- Brush radius

Learning goal:

- How geometry vertices work
- How brushes modify data

## Milestone 3 — Water and Biomes

Goal: start defining environments.

Features:

- Paint water
- Paint forest biome
- Paint grassland biome
- Paint coast or sand biome

Learning goal:

- Difference between object placement and data painting

## Milestone 4 — Places

Goal: add narrative identity.

Features:

- Create named places
- Select a region
- Add notes to a place

Learning goal:

- How project data is structured

## Milestone 5 — Scenes and Cameras

Goal: connect the world to storytelling.

Features:

- Save camera view
- Create scene entry
- Assign location
- Store time of day
- Store weather
- Store notes

Learning goal:

- How references avoid duplication

## Milestone 6 — Export for Flow

Goal: generate useful references.

Features:

- Export top view
- Export camera view
- Export scene description
- Export Flow prompt draft

Learning goal:

- How structured data becomes creative direction
