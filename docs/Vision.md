# Vision

Story Worlds Editor is a visual continuity editor for fictional environments.

It is designed for people who create stories, illustrations, comics, dream journals, visual novels or AI-generated scenes and need their places to remain consistent over time.

## What this project is

Story Worlds Editor is:

- A map and environment planning tool
- A visual story bible
- A reference system for consistent scenes
- A lightweight world editor
- A bridge between planning and final illustration

## What this project is not

Story Worlds Editor is not:

- A game
- A city builder
- A full 3D modeling program
- A replacement for Blender
- A renderer that must produce final artwork

The editor only needs to preserve structure, layout and continuity. Final images may be generated or illustrated separately.

## Main principle

The world is the source of truth.

A scene should not redefine the world. A scene should reference the world.

For example:

- The lake remains in the same place.
- The forest keeps its shape.
- The house remains beside the same path.
- A camera can return to the same viewpoint.
- A scene can remember time of day, weather and notes.

## Long-term goal

A user should be able to open a project months later and understand:

- Where each place is
- What each location means narratively
- Which scenes happen there
- What has changed over time
- How to describe the place for AI illustration

Story Worlds Editor should become a practical visual memory for stories.
